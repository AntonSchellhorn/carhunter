# 🚗 CarHunter Bot

Telegram-Bot zur automatischen Überwachung von Fahrzeugangeboten auf AutoScout24 und anderen Plattformen.

## Was macht der Bot?
- Sucht rund um die Uhr nach Fahrzeugen mit den gewünschten Parametern
- Vergleicht den Preis mit dem Marktdurchschnitt und bewertet das Angebot
- Sendet sofort eine Telegram-Benachrichtigung, wenn ein passendes Inserat gefunden wird

## Warum CarHunter?
Gute Angebote auf AutoScout24 sind oft innerhalb von Stunden weg. CarHunter schläft nicht — du verpasst kein Schnäppchen mehr.

## Technologie
- Python 3.11
- aiogram 3 — Telegram Bot Framework
- Playwright — Web Scraping
- APScheduler — Aufgabenplaner
- SQLite — Datenbank
- Docker + Hetzner VPS — Deployment
