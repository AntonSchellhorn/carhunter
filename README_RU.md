# 🚗 CarHunter Bot

Telegram-бот для автоматического мониторинга объявлений о продаже автомобилей на AutoScout24 и других платформах.

## Что делает
- Ищет автомобили по заданным параметрам 24/7
- Сравнивает цену с рыночной и оценивает выгоду
- Мгновенно отправляет уведомление в Telegram при нахождении подходящего объявления

## Стек
- Python 3.11
- aiogram 3 — Telegram Bot
- Playwright — парсинг сайтов
- APScheduler — планировщик задач
- SQLite — база данных
- Docker + Hetzner VPS — деплой
