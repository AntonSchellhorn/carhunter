# 🚗 CarHunter Bot

Telegram bot for automatic 24/7 monitoring of car listings on AutoScout24 and other platforms.

## What it does
- Searches for cars by your parameters around the clock
- Compares price with market average and evaluates the deal
- Instantly sends a Telegram notification when a matching listing is found

## Why CarHunter?
Good deals on AutoScout24 disappear within hours. CarHunter never sleeps — you won't miss a bargain again.

## Tech Stack
- Python 3.11
- aiogram 3 — Telegram Bot Framework
- Playwright — Web Scraping
- APScheduler — Task Scheduler
- SQLite — Database
- Docker + Hetzner VPS — Deployment
